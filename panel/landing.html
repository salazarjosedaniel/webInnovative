Cómo funciona
Flujo simple para el cliente y control total para el administrador. Sin puertas abiertas hacia la tienda: solo conexión segura hacia la nube.
🔁 Flujo operativo en 5 pasos
01
Escaneo
Lector USB 2D (HID). El cliente escanea el código del producto.
02
Consulta API
El visor consulta tu API/ERP/POS vía HTTPS (puerto 443).
03
Respuesta cloud
Respuesta JSON optimizada con precio, descripción, promos y stock (si aplica).
04
Visualización
Pantalla LCD 7" con UI rápida (producto / precio / promo / aviso).
05
Registro y control
Logs centralizados, estado del dispositivo, suscripción activa y métricas.
✓ Seguridad: HTTPS + validación de dispositivo antes de operar
✓ Resiliencia: watchdog + auto-reconexión + recuperación ante fallas
✓ Operación: actualizaciones OTA y monitoreo remoto de flota

No es solo una pantalla: es plataforma
Administra una o miles de unidades desde un panel web. Controla suscripción, firmware, estatus y métricas sin visitas técnicas.
📊Panel SaaS centralizado

Control de flota y operación en un solo lugar. Ideal para integradores y cadenas con múltiples sucursales.

Gestión de dispositivosAlta automática por DeviceId + IP + firmware, con “lastSeen”.
Suscripción y bloqueoValidación de pago antes de operar, sin intervención manual en tienda.
Logs y auditoríaRegistro de eventos para troubleshooting, calidad y soporte.
OTA / firmware remotoActualiza versiones, fuerza update y gestiona URLs de binarios.
AnalyticsConsultas por dispositivo, horarios pico y productos consultados (opcional).
🔒Seguridad y continuidad

Arquitectura “outbound-only”: sin abrir puertos hacia la tienda. Cifrado, validación y recuperación automática.

✓ HTTPS + TLS
✓ Zero-trust (dispositivo autenticado)
✓ Watchdog y auto-reconexión
✓ Sin acceso remoto directo a la red local
Integración flexible
Puedes integrar a tu API existente o usar un endpoint estándar. El visor solo necesita una URL y el formato de respuesta acordado.
